# Heartopia 像素画转换器

将图片转换为心动小镇游戏像素画矩阵。

## 功能

- 支持 5 种画布比例（16:9, 4:3, 1:1, 3:4, 9:16）
- 支持 4 个精细度等级（Level 0-3）
- 使用心动小镇游戏原生颜色（13 组 125 色，从原工程 color.svg 提取）
- 自动中心裁剪适应目标比例
- 自动处理 EXIF 旋转（手机照片）
- 导出 JSON 和 CSV 格式
- GUI 图形界面（需要 PySide6）

## 安装

```bash
pip install -r requirements.txt
```

如需使用 GUI 界面，还需安装：

```bash
pip install PySide6
```

## 使用方法

### 命令行

```bash
# 默认 1:1 比例，Level 2 精细度
python heartopia_converter.py image.jpg

# 指定比例和精细度
python heartopia_converter.py image.jpg 1:1 3
```

### GUI 界面

```bash
python gui.py
```

### 参数说明

- **比例**：
  - `16:9` - 宽屏（30x18, 50x28, 100x56, 150x84）
  - `4:3` - 标准（30x24, 50x38, 100x76, 150x114）
  - `1:1` - 正方形（30x30, 50x50, 100x100, 150x150）
  - `3:4` - 竖屏（24x30, 38x50, 76x100, 114x150）
  - `9:16` - 手机竖屏（18x30, 28x50, 56x100, 84x150）

- **精细度**：
  - `0` - 最低（最小网格，最快）
  - `1` - 低
  - `2` - 中（推荐）
  - `3` - 高（最大网格，最精细）

## 输出文件

### JSON 格式

```json
{
  "ratio": "1:1",
  "level": 2,
  "gridWidth": 100,
  "gridHeight": 100,
  "totalPixels": 8523,
  "colorCount": 45,
  "colors": {
    "#fece92": 234,
    "#a6263d": 189
  },
  "pixels": [
    {"x": 0, "y": 0, "color": "#fece92"},
    {"x": 1, "y": 0, "color": "#cf354d"}
  ]
}
```

### CSV 格式

```csv
x,y,color
0,0,#fece92
1,0,#cf354d
```

## 颜色系统

使用从 [原工程](https://github.com/zerochansy/Heartopia-Painting-Tools) 的 `color.svg` 中提取的 125 种游戏原生颜色，分为 13 组：

| 组号 | 名称 | 主色 |
|------|------|------|
| 1 | 黑白灰 | 5 色 |
| 2 | 红色系 | 10 色 |
| 3 | 橙红色系 | 10 色 |
| 4 | 橙色系 | 10 色 |
| 5 | 黄色系 | 10 色 |
| 6 | 黄绿色系 | 10 色 |
| 7 | 绿色系 | 10 色 |
| 8 | 青绿色系 | 10 色 |
| 9 | 青色系 | 10 色 |
| 10 | 蓝色系 | 10 色 |
| 11 | 蓝紫色系 | 10 色 |
| 12 | 紫色系 | 10 色 |
| 13 | 粉色系 | 10 色 |

## 注意事项

- 图片会自动按目标比例中心裁剪，再缩放到网格尺寸
- 颜色通过 RGB 欧几里得距离自动匹配到最接近的游戏原生颜色
- 手机拍摄的照片会自动处理 EXIF 旋转信息
