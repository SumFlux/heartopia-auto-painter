# Heartopia 像素画转换器

将图片转换为心动小镇游戏像素画矩阵，供 [自动画画脚本](../painter/README.md) 使用。

## 功能

- 5 种画布比例（16:9、4:3、1:1、3:4、9:16）
- 4 个精细度等级（Level 0-3）
- 游戏原生 126 色调色板精确匹配（数据来源：`shared/palette.py`）
- Floyd-Steinberg 抖动算法（可选，更平滑的颜色过渡）
- 图像增强预处理（饱和度、对比度、锐度可调）
- 自动中心裁剪适应目标比例
- 自动处理 EXIF 旋转（手机照片）
- 导出 JSON（含 `colorId` 定位标识）和 CSV 格式
- GUI 图形界面（实时预览 + 参数调节）

## 安装

```bash
pip install -r requirements.txt
```

`requirements.txt` 包含：Pillow、NumPy、PySide6。

## 使用方法

### GUI 界面（推荐）

```bash
python gui.py
```

1. 点击「导入图片」选择图片文件
2. 选择画布比例和精细度等级
3. 可选开启「抖动」和「增强」，调节增强参数
4. 实时预览效果
5. 点击「导出 JSON」保存像素数据

### 命令行

```bash
# 默认 1:1 比例，Level 2 精细度
python heartopia_converter.py image.jpg

# 指定比例和精细度
python heartopia_converter.py image.jpg 1:1 3

# 指定输出路径
python heartopia_converter.py image.jpg 16:9 2 -o output.json
```

## 参数说明

### 画布比例与网格尺寸

| 比例 | Level 0 | Level 1 | Level 2 | Level 3 |
|:----:|:-------:|:-------:|:-------:|:-------:|
| 16:9 | 30×18 | 50×28 | 100×56 | 150×84 |
| 4:3 | 30×24 | 50×38 | 100×76 | 150×114 |
| 1:1 | 30×30 | 50×50 | 100×100 | 150×150 |
| 3:4 | 24×30 | 38×50 | 76×100 | 114×150 |
| 9:16 | 18×30 | 28×50 | 56×100 | 84×150 |

### 图像增强参数

| 参数 | 默认值 | 说明 |
|------|:------:|------|
| 饱和度 | 1.3 | >1 更鲜艳，<1 更灰 |
| 对比度 | 1.2 | >1 更分明，<1 更柔和 |
| 锐度 | 1.3 | >1 更锐利，<1 更模糊 |

## 输出文件格式

### JSON

```json
{
  "ratio": "1:1",
  "level": 2,
  "gridWidth": 100,
  "gridHeight": 100,
  "totalPixels": 8523,
  "colorCount": 45,
  "colors": {
    "#fece92": 234,
    "#a6263d": 189
  },
  "pixels": [
    {"x": 0, "y": 0, "color": "#fece92", "colorId": "3-3"},
    {"x": 1, "y": 0, "color": "#cf354d", "colorId": "1-0"}
  ]
}
```

**关键字段说明**：

| 字段 | 说明 |
|------|------|
| `gridWidth` / `gridHeight` | 像素矩阵的宽高 |
| `totalPixels` | 需要绘制的非背景像素总数 |
| `color` | 匹配到的游戏颜色（hex） |
| `colorId` | 游戏调色板定位标识，格式 `"组号-组内索引"`，如 `"1-0"` 表示第 2 组第 1 个色 |

> `colorId` 是自动画画脚本的核心依据。它告诉脚本需要把调色板翻到第几组、点第几个色块，避免了颜色匹配误差。

### CSV

```csv
x,y,color,colorId
0,0,#fece92,3-3
1,0,#cf354d,1-0
```

## 颜色系统

使用游戏截图实际取色的 126 种原生颜色，统一定义在 `shared/palette.py` 中：

- 第 0 组（黑白灰）：6 色
- 第 1-12 组（红→粉）：各 10 色

颜色匹配算法：RGB 欧几里得距离（平方距离，无需开方）。

## 注意事项

- 图片会自动按目标比例中心裁剪后缩放到网格尺寸
- 抖动模式会让颜色更自然，但可能增加使用的颜色种类数（绘制时间更长）
- 增强模式在缩放前预处理，对低精细度（Level 0-1）效果更明显
- 背景色 `#feffff` 会被标记为透明，自动画画时跳过不画
