# Heartopia 自动画画脚本

读取 [转换器](../converter/README.md) 导出的像素矩阵 JSON，自动在心动小镇中选色并绘制。

## 功能

- GUI 控制面板（PySide6）
- 导入 JSON 像素数据
- 画布 4 角标定
- 调色板标定（左右标签 + 色块区域）
- 自动检测画布 4 角标记点
- 固定坐标保存 / 自动应用
- 工具栏标定（画笔 + 油漆桶）
- 油漆桶填充优化
- 快捷键控制：F5 / F6 / F7
- 断点续画

## 环境要求

- Windows
- Python 3.8+
- 心动小镇正在运行，且已进入画画界面

## 安装与启动

```bash
pip install -r requirements.txt
python auto_painter.py
```

也可以双击 `auto_painter_main.bat`。

## 当前界面流程

### 1. 导入 JSON

点击「导入 JSON 文件」。

导入后日志会显示：
- 画布尺寸
- 像素数量
- 是否包含 `colorId`

说明：
- 有 `colorId`：优先按精确定位绘制
- 没有 `colorId`：退回最近邻匹配，可能有轻微偏差

### 2. 标定画布（4 次 Enter）

按钮：`标定画板范围（4 个角，共 4 次 Enter）`

按提示在游戏中依次标定：
1. 左上
2. 右上
3. 左下
4. 右下

这是当前实现，不再是旧文档里的 2 点标定。

### 3. 自动检测画布（可选）

按钮：`自动检测画布（4角标记点）`

用法：
- 先在游戏画布四角各放一个明显颜色的标记点
- 再点击按钮自动检测

适合不想手动逐点标定时使用。

### 4. 标定调色板（4 次 Enter）

按钮：`标定调色板（左右标签 + 色块区域，共 4 次 Enter）`

依次标定：
1. 色系标签最左
2. 色系标签最右
3. 色块区域左上第一格
4. 色块区域右下最后一格

程序会自动推导 2×5 色块区域内的全部色块位置。

### 5. 固定坐标（推荐）

相关按钮：
- `固定当前坐标`
- `从窗口自动标定`
- `清除固定坐标`

推荐流程：
1. 完成一次手动标定
2. 点击「固定当前坐标」保存相对窗口坐标
3. 下次打开后可直接点「从窗口自动标定」

这样能减少重复标定。

### 6. 工具栏标定（油漆桶功能需要）

按钮：`标定工具栏（画笔+油漆桶）`

需要标定两个位置：
1. 画笔工具
2. 油漆桶工具

如果没有这一步，油漆桶优化不可用。

### 7. 开始画画

按钮：`开始画画 (F5)`

开始前可设置：
- 绘画速度
- 是否启用油漆桶填充
- 微调偏移 X / Y

绘制过程中会显示：
- 当前进度
- 当前颜色组
- 颜色预览
- 日志输出

### 8. 暂停 / 停止 / 续画

| 快捷键 | 功能 |
|:------:|------|
| F5 | 开始 |
| F6 | 暂停 / 恢复 |
| F7 | 停止 |

按钮：`断点续画`

当存在 `paint_progress.json` 时，可以从中断位置继续。

## 油漆桶填充说明

界面选项：`油漆桶填充（大面积同色区域自动填充）`

当前实现：
- 仅在同色 4 连通区域足够大时启用
- 先画边界，再用油漆桶填充内部
- 需要已保存固定坐标，并且包含工具栏位置

如果未满足条件，程序会退回普通逐点绘制。

## 速度预设

| 预设 | 点击间隔 |
|------|:--------:|
| 快速 | 20ms |
| 正常 | 50ms |
| 慢速 | 100ms |
| 极慢 | 200ms |

补充：切换颜色本身仍有固定等待时间，速度不会无限提升。

## 文件说明

| 文件 | 作用 |
|------|------|
| `auto_painter.py` | GUI 主入口 |
| `paint_engine.py` | 绘画编排、分组、断点续画、油漆桶策略 |
| `palette_navigator.py` | 调色板翻页与选色 |
| `canvas_locator.py` | 画布像素坐标到屏幕坐标映射 |
| `mouse_input.py` | 输入后端 |
| `window_manager.py` | 游戏窗口查找与截图辅助 |
| `config.py` | painter 配置 |
| `calibration.json` | 标定数据 |
| `fixed_positions.json` | 固定坐标数据 |
| `paint_progress.json` | 断点续画进度 |

## 故障排查

| 问题 | 可能原因 | 解决方式 |
|------|---------|---------|
| 点击偏移 | 标定不准 / DPI / 多显示器 | 重新标定，必要时用微调偏移 |
| 颜色选错 | 调色板标定不准，或 JSON 不含 `colorId` | 重做调色板标定，优先使用带 `colorId` 的 JSON |
| 不能启用油漆桶 | 没有固定坐标或工具栏位置 | 先保存固定坐标，再标定工具栏 |
| 自动标定失败 | 固定坐标或窗口状态不匹配 | 重新手动标定并重新保存固定坐标 |
| 断点续画不可用 | 没有进度文件 | 先启动绘制并正常停止一次 |
| 游戏不响应点击 | 窗口失焦或被遮挡 | 保持游戏处于前台可操作状态 |
